# 📋 Codec Technologies - Project Management Tool

A full-stack MERN Project Management Tool developed during the Codec Technologies MERN Stack Internship.

## 🚀 Features

- Create Tasks
- Update Tasks
- Delete Tasks
- Task Status Management
- Assign Tasks to Team Members
- Due Date Management
- Task Dashboard
- Task Statistics
- Responsive UI

## 🛠 Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Axios

## 📂 Project Structure

```
backend/
frontend/
```

## ⚙️ Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
